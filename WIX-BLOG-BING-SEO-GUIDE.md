## 第一步：注册 Wix 账号

### 1.1 访问 Wix 官网

🔗 打开浏览器访问：**https://www.wix.com**

### 1.2 注册账号

有三种注册方式：

- **方式一：邮箱注册**
  - 在首页输入你的邮箱地址
  - 点击 "Get Started" 或 "开始创建"
  - 设置密码（至少 8 个字符）

- **方式二：Google 账号登录**（推荐，最快）
  - 点击 "Continue with Google"
  - 选择你的 Google 账号

- **方式三：Facebook 账号登录**
  - 点击 "Continue with Facebook"

### 1.3 完成注册
- 填写简单的调查问卷（可选，可以跳过）
- 选择 "I'm new to building websites"（如果是新手）

## 第二步：创建博客网站

### 2.1 选择创建方式

Wix 会询问你如何创建网站，选择以下之一：

**选项 A：AI 网站生成器（最快，推荐新手）**
- 点击 "Create with AI"
- 回答几个问题（网站类型选 "Blog"）
- AI 会自动生成一个博客模板
- 预计时间：2-3 分钟

**选项 B：模板编辑器（自定义程度高）**
- 点击 "Choose a template"
- 搜索 "Blog" 或 "博客"
- 浏览并选择一个喜欢的模板
- 点击 "Edit" 开始编辑

### 2.2 推荐的博客模板

以下是几个适合个人博客的模板：

1. **Minimalist Blog** - 简约现代风格 ⭐⭐⭐⭐⭐
2. **Tech Blog** - 适合技术博客 ⭐⭐⭐⭐⭐
3. **Personal Journal** - 个人日记风格 ⭐⭐⭐⭐
4. **Magazine Style** - 杂志风格，适合多内容 ⭐⭐⭐⭐

### 2.3 命名你的网站

- 在弹出的对话框中输入网站名称
- 例如：`Renomee AI Blog`、`科技前沿` 等
- 点击 "Continue"

## 第三步：设计博客

### 3.1 编辑器界面说明

进入 Wix 编辑器后，你会看到：

- **左侧栏**：添加元素（文本、图片、按钮等）
- **顶部栏**：页面管理、设置、预览、发布
- **中间区域**：网站预览（可以直接点击编辑）
- **右侧栏**：元素属性设置

### 3.2 基础设置

#### 修改网站标题
1. 点击页面顶部的标题文字
2. 输入你的博客名称（如 "Renomee AI"）
3. 调整字体、颜色、大小

#### 修改导航菜单
1. 点击顶部菜单栏
2. 点击 "Manage Menu"
3. 添加/删除/重命名菜单项：
   - Home（首页）
   - Blog（博客）
   - About（关于）
   - Contact（联系）

#### 修改配色方案
1. 点击左侧 "Design" 或 "设计"
2. 选择 "Site Colors"
3. 选择预设配色或自定义颜色

#### 修改字体
1. 点击左侧 "Design"
2. 选择 "Site Fonts"
3. 选择合适的字体组合（建议：标题用粗体，正文用易读字体）

### 3.3 添加博客部分

如果模板没有博客功能：

1. 点击左侧 "Add" 或 "+"
2. 选择 "Blog"
3. 拖拽 "Blog Feed" 到页面
4. 调整位置和大小

## 第四步：发布第一篇文章

### 4.1 进入博客管理

1. 在编辑器中，找到并点击博客模块
2. 点击 "Manage Posts" 或 "管理文章"
3. 或者点击顶部菜单 → "Blog" → "Manage Blog"

### 4.2 创建新文章

1. 点击 "Create New Post"
2. 填写文章信息：

```
标题：深度测评滴答修免费图片转PDF vs 国内工具价格大起底

正文：
（粘贴你已有的文章内容，或者重新写）

特色图片：
- 点击 "Add Cover Image"
- 上传一张吸引人的封面图（建议尺寸：1200x628px）

摘要：
在这个数字化时代，我们每天都在与各种图片打交道...
（写一段 150-200 字的摘要）
```

### 4.3 SEO 设置（文章级别）

在文章编辑器中，找到 "SEO Settings"：

1. **SEO 标题**（最重要）
   ```
   深度测评：滴答修免费图片转PDF工具 vs 国内付费工具价格对比
   ```

2. **SEO 描述**（160 字符以内）
   ```
   详细对比滴答修免费图片转PDF工具与国内主流付费工具。实测数据、价格分析、使用体验一文全解。适合需要图片转PDF的个人和企业用户。
   ```

3. **URL Slug**（网址后缀）
   ```
   image-to-pdf-tools-comparison-2025
   ```

### 4.4 发布文章

1. 点击 "Publish" 按钮
2. 等待几秒钟
3. 文章发布成功 ✅

## 第五步：配置 SEO 设置

### 5.1 全局 SEO 设置

1. 在编辑器顶部，点击 **"Settings"（设置）**
2. 选择 **"SEO (Google)"** 或 **"Marketing & SEO"**

### 5.2 首页 SEO

```
网站标题：
Renomee AI - 深度测评与科技工具分享

网站描述：
专注于科技工具深度测评、生产力软件推荐、AI应用分享。提供真实使用体验和详细价格对比，帮助你找到最适合的工具。

关键词（可选）：
图片转PDF, 在线工具, 软件测评, 生产力工具, AI工具
```

### 5.3 社交分享设置

1. 在 SEO 设置中找到 "Social Share"
2. 上传社交分享图片（建议 1200x630px）
3. 设置 Facebook 和 Twitter 分享标题和描述

### 5.4 发布网站

⚠️ **重要：必须先发布网站才能验证 Bing**

1. 点击右上角蓝色 **"Publish"** 按钮
2. 选择免费域名：
   ```
   例如：renomeeai.wixsite.com/blog
   或：yourusername.wixsite.com/yoursite
   ```
3. 点击 "Publish Now"
4. 等待发布完成（约 30-60 秒）
5. 记下你的网站地址 📝

## 第六步：验证 Bing 站长工具

### 6.1 注册 Bing Webmaster Tools

🔗 访问：**https://www.bing.com/webmasters**

1. 点击 "Sign in" 或 "登录"
2. 使用 Microsoft 账号登录（或注册新账号）

### 6.2 添加网站

1. 点击 "Add a Site" 或 "添加站点"
2. 输入你的 Wix 网站地址：
   ```
   https://yourusername.wixsite.com/yoursite
   ```
3. 点击 "Add"

### 6.3 验证网站（Wix 免费版可行方案）

⚠️ **重要提示**：Wix 免费版的限制
- ❌ Custom Code（自定义代码）需要付费
- ❌ 无法上传自定义文件到根目录  
- ❌ DNS 验证需要自定义域名

#### ✅ 方式一：通过 Wix SEO 设置添加 Meta 标签（免费可用）

1. Bing 会显示一个 meta 标签，类似：
   ```html
   <meta name="msvalidate.01" content="XXXXXXXXXXXXXXXXXXXXXXXX" />
   ```

2. **复制验证码**（content 里面的内容）：`XXXXXXXXXXXXXXXXXXXXXXXX`

3. 回到 Wix 编辑器：
   - 点击左侧 "Settings"（设置）
   - 选择 "SEO (Google)"
   - 向下滚动找到 **"Verify Your Site with Google"**
   - 虽然标题说的是 Google，但这里可以输入任何验证 meta 标签

4. 如果上述方法不行，尝试这个：
   - 在 Wix 编辑器中，点击任意页面
   - 点击页面设置（齿轮图标）
   - 选择 "SEO (Google)"
   - 找到 "Advanced SEO" → "Additional Tags"
   - 粘贴完整的 meta 标签

5. **重新发布网站**（点击右上角 "Publish"）

6. 等待 1-2 分钟后，回到 Bing Webmaster Tools

7. 点击 "Verify" 验证

#### ✅ 方式二：XML 文件验证（需要 Wix 付费计划）

**仅适用于以下付费计划：Combo, Unlimited, VIP, Business, Ecommerce**

如果你愿意升级（最便宜的 Combo 计划约 $16/月）：

**步骤 A：下载验证文件**
1. 在 Bing Webmaster Tools，选择 "XML File" 验证方式
2. 下载文件，例如：`BingSiteAuth.xml`

**步骤 B：上传到 Wix**

Wix 不支持直接上传到根目录，但可以这样做：

1. 在 Wix 编辑器，点击左侧 "Media" → "Site Files"
2. 上传 `BingSiteAuth.xml`
3. 文件会被放在：`https://yourusername.wixsite.com/yoursite/_files/ugd/xxxxx/BingSiteAuth.xml`

⚠️ **问题**：这个路径不是根目录，Bing 可能无法识别

**更好的方法（需要付费版的 Velo 开发模式）：**

1. 在 Wix 编辑器顶部，启用 "Dev Mode"（开发者模式）
2. 在左侧文件管理器中，找到 "Public" 文件夹
3. 将 `BingSiteAuth.xml` 上传到 Public 文件夹
4. 文件会在：`https://yourusername.wixsite.com/BingSiteAuth.xml`
5. 回到 Bing，点击 "Verify"

#### ❌ 方式三：DNS CNAME 验证（不适用于 Wix 子域名）

仅适用于已连接自定义域名（如 renomeeai.com）的网站。

## 第七步：配置 IndexNow

### 7.1 什么是 IndexNow？

IndexNow 是一个允许网站即时通知搜索引擎内容更新的协议。**Wix 已经自动集成了 IndexNow**，无需手动配置！

### 7.2 确认 IndexNow 已启用

1. 在 Wix 编辑器中，进入 "Settings" → "SEO"
2. 查看是否有 "IndexNow" 选项
3. 如果有，确保已启用 ✅

### 7.3 Wix 的自动 IndexNow

Wix 会在以下情况自动触发 IndexNow：
- ✅ 发布新文章
- ✅ 更新现有文章
- ✅ 修改页面内容
- ✅ 发布网站更新

**你不需要做任何操作！** 🎉

## 第八步：提交 URL 到 Bing

### 8.1 ⭐ 免费用户推荐：使用 Bing URL Submission Tool（无需验证）

**最简单的方法 - 不需要验证网站！**

🔗 访问：**https://www.bing.com/webmasters/url-submission**

**优点：**
- ✅ 完全免费
- ✅ 无需验证网站所有权
- ✅ 立即提交，快速索引

**限制：**
- ⚠️ 每天最多提交 10 个 URL
- ⚠️ 需要手动提交

**操作步骤：**

1. 访问 Bing URL 提交工具
2. 输入你的 URL（一次最多 10 个），例如：
   ```
   https://yourusername.wixsite.com/yoursite
   https://yourusername.wixsite.com/yoursite/post/your-first-article
   https://yourusername.wixsite.com/yoursite/about
   ```
3. 输入验证码
4. 点击 "Submit"
5. 完成！✅

**建议提交顺序（前 10 个重要页面）：**
1. 首页
2. 博客列表页
3. 最新的 3-5 篇文章
4. 关于页面
5. 联系页面

**每天提交策略：**
- 第 1 天：提交首页 + 核心页面
- 第 2 天：提交最新文章
- 第 3 天：提交其他文章
- 持续每天提交，直到所有页面都提交完

### 8.2 已验证用户：通过 Sitemap 自动提交

**仅适用于已成功验证 Bing Webmaster Tools 的用户**

1. 在 Bing Webmaster Tools 中，找到 **"Sitemaps"**
2. Wix 自动生成的 Sitemap 地址：
   ```
   https://yourusername.wixsite.com/yoursite/sitemap.xml
   ```
3. 点击 "Submit a Sitemap"
4. 输入：`/sitemap.xml`
5. 点击 "Submit"

### 8.3 已验证用户：手动提交单个 URL

如果你已验证网站，可以在 Webmaster Tools 中提交：

1. 在 Bing Webmaster Tools 左侧，点击 "URL Submission"
2. 输入要提交的 URL（一次最多 10,000 个）
3. 点击 "Submit"

## 🔧 常见问题

### Q1: Wix 免费版有什么限制？

**A:** 主要限制：
- 子域名（yourname.wixsite.com）
- 显示 Wix 广告
- 500MB 存储
- 不能连接自定义域名

**但是**：可以正常使用 Bing 站长工具和 IndexNow ✅

### Q2: 多久能在 Bing 搜索到我的网站？

**A:** 通常需要：
- 提交后 **24-48 小时**：Bing 开始爬取
- **3-7 天**：出现在搜索结果
- **2-4 周**：排名稳定

### Q3: 如何检查网站是否被 Bing 收录？

**A:** 在 Bing 搜索框输入：
```
site:yourusername.wixsite.com
```
如果显示结果，说明已被收录。

### Q4: 可以同时提交到 Google 吗？

**A:** 当然可以！Wix 同样支持 Google Search Console：
1. 访问 https://search.google.com/search-console
2. 添加网站（验证方式与 Bing 类似）
3. 提交 Sitemap

### Q5: IndexNow 会自动提交所有更新吗？

**A:** 是的！在 Wix 上：
- 发布新文章 → 自动通知 Bing
- 更新文章 → 自动通知 Bing
- 无需手动操作 ✅

### Q6: Wix 免费版真的无法验证 Bing 站长工具吗？

**A:** 是的，Wix 免费版有限制：
- ❌ Custom Code（添加 Meta 标签）需要付费
- ❌ 无法上传 XML 文件到根目录
- ❌ 子域名无法做 DNS 验证

**解决方案：**
- ✅ 使用 Bing URL Submission Tool（无需验证）
- ✅ 依靠 Wix 自动 IndexNow（已内置）
- ✅ 或升级到 Combo 计划（$16/月）

### Q6.1: 不验证网站，Bing 还会收录我的网站吗？

**A:** 会的！验证只是为了：
- 查看详细统计数据
- 手动提交 Sitemap
- 使用高级 SEO 工具

**但 Bing 依然会：**
- ✅ 自动爬取你的网站
- ✅ 索引你的内容
- ✅ 在搜索结果中显示

**Wix 已经帮你做了：**
- ✅ 自动生成 Sitemap
- ✅ 自动 IndexNow 提交
- ✅ 优化 SEO 结构

### Q7: 如何迁移 WordPress.com 的内容到 Wix？

**A:** 
1. 在 WordPress.com 导出内容（Settings → Export）
2. 在 Wix 博客管理中逐篇复制粘贴
3. 或使用 Wix 的导入工具（仅付费版）
